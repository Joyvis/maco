# MVP - Life Planner App

## Overview
Initial release focusing on the financial module with core features for tracking money, planning goals, and managing bills.

## Scope
Two core features for MVP:

1. **Track money usage**: 
   - Record income and expenses (manual entry for MVP)
   - Report page showing total earnings and spending
   - Category breakdown visualization (pie chart) showing spending distribution (e.g., x% taxes, x% health, x% leisure)
   - Recurring transaction schedules with cost projections

2. **Plan goals**: Create savings goals with target dates and budgets. The app calculates required monthly savings to reach goals

**Note:** Bill management will be addressed when payment methods are introduced in a future version.

## Success Criteria
User can track income/expenses, view spending reports with category breakdown, set up recurring schedules with projections, and create a savings goal.
